# Coordination Rotation

### Coordinate rotation

```
   x1 = x * Math.cos(rotation) – y * Math.sin(rotation);
   y1 = y * Math.cos(rotation) + x * Math.sin(rotation);
```

### Reverse coordinate rotation
   
```
   x1 = x * Math.cos(rotation) + y * Math.sin(rotation);
   y1 = y * Math.cos(rotation) – x * Math.sin(rotation);
```
